# MizpaLive MVP — Plan de Producto

## ¿Qué es Mizpa?

**Mizpa** es un agente de IA que analiza sitios web y genera arquitecturas React modernas basándose en el diseño y contenido existente del sitio.

En términos simples: **pegás una URL, Mizpa analiza tu web, te dice qué está mal, y te genera una versión mejorada en React + Tailwind.**

---

## Problema que resuelve

Crear una web moderna desde cero es difícil:
- No sabés qué componentes generar
- No extraés los design tokens (colores, tipografía, espaciado) del sitio actual
- No sabés qué problemas SEO tenés
- Generar código React limpio toma mucho tiempo

**Mizpa automatiza todo esto.** Vos solo pegás la URL, y Mizpa hace el trabajo pesado.

---

## Usuario Objetivo

- **Freelancers** que quieren modernizar sitios de clientes rápido
- **Dev teams** que necesitan un boilerplate base de un sitio existente
- **Emprendedores** que tienen un sitio viejo y quieren migrar a React
